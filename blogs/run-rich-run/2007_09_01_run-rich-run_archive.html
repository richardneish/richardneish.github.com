Monday, September 03, 2007

Work to home - 10.6km

Tube strike today, so ran home from Bishopsgate to St John's Wood, via Regent's Canal. Haven't done this route before, so had a few wrong turns around Angel before linking up with the canal, but overall very smooth. Will definitely do again.

Haven't done any activity for ages, so took it slow and easy, stopping to stretch hamstrings at the start of the run, then sticking to slow and easy pace. Minor chafing issues, but overall very pleased with this.

Nottingham half-marathon in less than two weeks, and I haven't done any long runs, so that will be messy, but if I stick to this slow, easy pace, I should get round injury-free. No idea what that time will look line...

Watch this space.

Distance: 10.6km
Time: 1:01:27
Pace: 5'48" per km
Terrain: flat on roads and canal towpath
